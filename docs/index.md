# AnchorsAway

Base URL anchor fix.

### Requirements

* MODX Revolution 2.8+
* PHP 7.2+

### Features

* Makes anchor links on the same page behave as if the base url tag is omitted
* Add a data-anchor attribute by option for use with javascript
